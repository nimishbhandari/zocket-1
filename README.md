# Setup

### Make a .env file in root folder with the following

```
PORT=5000
MODE=Development
MONGO_URI=
```

### Install server dependencies

```bash
npm install
```

### Install frontend dependencies

```bash
cd frontend
npm install
```

### Run Server Express

```bash
yarn start
```

### Run Server ReactJS

```bash
cd frontend
yarn start
```

### Build for production

```bash
cd frontend
npm run build
```
